A beautiful, functional, and cross-platform Pomodoro timer built with Flutter and Material 3 design.

## ‚ú® Key Features

- **Pomodoro Cycles**: Customizable focus and rest durations to match your productivity style.
- **Material 3 Design**: A modern, clean, and intuitive user interface.
- **Cross-Platform**: Available for both **Windows** and **Android**.
- **Compact Mode**: A minimalistic floating timer for Windows that stays out of your way while you work.
- **Statistics**: Detailed visualization of your focus time (Daily, Weekly, Monthly, Yearly).
- **Cloud Sync**: Sync your focus data across devices using Firebase Authentication and Firestore.
- **Desktop Notifications**: Native Windows notifications to alert you when sessions start/end.

## üöÄ Installation

### Windows
1. Go to the [Releases](https://github.com/earify/potato-timer/releases) page.

1. Go to the [Releases](https://github.com/earify/potato-timer/releases) page.
2. Download the latest `potato-android.apk`.
3. Install the APK on your device (you may need to allow installation from unknown sources).
